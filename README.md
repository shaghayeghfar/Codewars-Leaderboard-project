# Codewars Leaderboard Project

A simple leaderboard website that fetches user scores from the Codewars API and displays them.

**Features:**

- Enter a list of Codewars usernames
- Fetch scores and show a table with Username, Clan, and Score
- Select language or Overall ranking
- Top scorer highlighted
- Handles invalid users and API/network errors

**Technologies:**

- HTML, CSS, JavaScript
- Jest for unit testing
